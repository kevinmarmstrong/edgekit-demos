Golden demo: public-site-agent-user

A public site with a governed agent user.

This small public site shows Edgekit as an app-level agent user, not a generic chat widget. The host app owns the knowledge source, identity copy, capability-mode UI, telemetry, and refusal policy.

Capability mode

Checking browser-local capability before enabling the agent user.

Configured assistant identity

The assistant is Edgekit Public Site Guide, a read-only guide for this demo site. It answers from configured runtime identity and site evidence, not unsupported model memory.

Strict grounding

The agent must call searchPublicSite before factual answers, cite supporting site sections, and refuse unsupported identity, location, biography, hardware, affiliation, or disambiguation claims.

Basic fallback is labeled

If no browser-local model is available, the no-model path returns a visibly labeled Basic fallback answer using the same read-only site search tool. It is not presented as the full product experience.
